What is Kotlin?

A statically-typed JVM languaged developed by JetBrains

  • Development started in 2010
  • Current release: 1.0 Beta 4 (Dec. 22, 2015)

Features

  • Concise syntax
  • Compiles to JVM bytecode or Javascript
  • Interoperates seamlessly with Java
  • Null safety
  • A lot more!

Syntax


package hello

fun main(args: Array) : Unit {
   println("Hello World!")
}

or more simply:


package hello

fun main(args: Array) {
   println("Hello World!")
}

Class Syntax


class Greeter(val name: String) {
   fun greet() {
      println("Hello, $name")
   }
}

fun main(args: Array) {
   Greeter(args[0]).greet()
}

Variables

Immutable variables


val a: Int = 1
val b = 1   // `Int` type is inferred
val c: Int  // Type required when no initializer is provided
c = 1       // definite assignment

Mutable variables


var x = 5 // `Int` type is inferred
x += 1

String Templates


val i = 10
val s = "i = $i" // evaluates to "i = 10"

expressions are supported via curly braces:


val s = "abc"
val str = "$s.length is ${s.length}" // "abc.length is 3"
